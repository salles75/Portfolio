# Imagens do Projeto Eclat E-commerce

Adicione as imagens do projeto nesta pasta com os seguintes nomes:

- **HOME.png** - Página principal (imagem de capa)
- **PRODUCTS.png** - Página de produtos
- **CART.png** - Carrinho de compras
- **CHECKOUT.png** - Página de checkout

## Como adicionar as imagens:

1. Copie as imagens do seu projeto eclat-ecommerce da pasta `public/img/eclat/`
2. Cole nesta pasta: `public/projects/eclat/`
3. Renomeie as imagens seguindo os nomes acima (se necessário)

**Importante:** A imagem HOME.png será usada como capa principal do projeto no portfólio.
